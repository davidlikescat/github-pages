2. 언어 검사 피드백

실시 검사
수용(이해) 언어
표현(말하기) 언어
어휘력
전반적 언어 발달
채점 및 보고서 추후 전달
검사 중 관찰
관찰 항목내용평가
집중력 40분간 착석하여 검사 수행 양호
동기·의욕 당일 컨디션 저하 영향 낮음
모르는 문항 "모르겠어"라고 말하지 않고 입을 닫음 침묵 거부
아는 문항 말이 많아지고 자발적으로 이야기 만들어 표현 활발
문법 형태소 조사(와, 에, 를 등) 생략 나타남 미흡
단어 수준 단어 자체의 생략은 없음 정상
모방 선생님·TV 대사 문장 통째 학습 재현 우수
언어 치료사 소견
  • 말 시작이 늦어 또래 대비 문장 표현이 아직 매끄럽지 않음
  • 화용(pragmatics) 발달이 가장 미흡 — 상황에 맞는 말 사용 어려움
  • 좋아하는 주제에서는 풍부한 표현 가능
  • 모르거나 어려운 상황에서 말하기를 거부하는 패턴 → 또래 관계에서도 나타날 수 있음
핵심: "엄마와 아빠는 오늘 백화점에 갔다" → 다은이 표현: "엄마 아빠는 오늘 백화점에 갔다" (조사 '와' 생략)

참고: 당일 컨디션 저하(구토 3회)로 후반부 집중력이 떨어졌으나 전반적으로 잘 수행. 후반부 힘들어하며 엎드리는 모습 보였으나 시각 관련 항목에서는 실력 발휘.

강점과 개선 필요 영역

강점

  • 규칙·규율 잘 따름
  • 모방 능력 우수
  • 시지각 능력 양호~우수

  • 성격이 크게 예민하지 않음

개선 필요

  • 화용 언어 (상황 맞는 대화)
  • 양방향 소통
  • 모르는 것 표현하기
  • 문법 형태소(조사) 사용
  • 일상생활 기술 (옷 입기 등)
  • 시각 추적 (정중선 교차)
